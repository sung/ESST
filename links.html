Programs

Baton: a structure alignment program
CRESCENDO: a program for predicting functional residues from a three dimensional structure
Subst: a program for making ESST
Ulla: a new program for ESST generation
JOY: a program to annotate protein sequence alignments with three-dimensional (3D) structural features.
Kin3DCont: produce contour maps in kinemage format

Databases

HOMSTRAD: Homologous Structure Alignment Database
SCOP: Structural Classification Of Proteins
CSA: Catalytic Site Atlas
UniProt: a database providing comprehensive, high-quality and freely accessible resource of protein sequence and functional information
InterPare: a database server for protein interaction interface information
